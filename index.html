Extremos locais

Máximos e mínimos em vizinhanças — com o Teste da Derivada Segunda

Mínimo local

f(x₀,y₀) é mínimo local se existir um disco aberto D centrado em (x₀,y₀) tal que:

f(x₀,y₀) ≤ f(x,y), ∀(x,y) ∈ D

A superfície forma uma "bacia" ao redor do ponto.

Máximo local

f(x₀,y₀) é máximo local se existir um disco aberto D centrado em (x₀,y₀) tal que:

f(x₀,y₀) ≥ f(x,y), ∀(x,y) ∈ D

A superfície forma um "morro" ao redor do ponto.

⚠️ O extremo local é o valor f(x₀,y₀), não o ponto (x₀,y₀) nem o ponto do gráfico (x₀,y₀,f(x₀,y₀)).

Ponto crítico Definição 3

P = (x₀,y₀) é ponto crítico de f(x,y) se:

fₓ(x₀,y₀) = 0 E f_y(x₀,y₀) = 0
— ou —
fₓ(x₀,y₀) não existe ou f_y(x₀,y₀) não existe

✔ Teorema 1: todo extremo local ocorre em um ponto crítico. A recíproca não é verdadeira.

Ponto de sela Definição 4

Um ponto crítico que não é extremo. Na vizinhança existem pontos com f maior e pontos com f menor — como uma sela de cavalo.

Exemplo clássico:
f(x,y) = y² − x² → (0,0) é ponto de sela

f(x,0) = −x² < 0 // desce no eixo x
f(0,y) = y² > 0 // sobe no eixo y
Discriminante / Hessiano Definição 5
D(x,y) = fₓₓ · f_yy − (fₓ_y)²

É o determinante da matriz Hessiana de f. Mede a "concavidade combinada" da função no ponto crítico.

Teste da derivada segunda Teorema 2
D(x₀,y₀)fₓₓ(x₀,y₀)Conclusão
D < 0—Ponto de sela
D > 0fₓₓ > 0Mínimo local
D > 0fₓₓ < 0Máximo local
D = 0—Inconclusivo
Passo a passo — encontrar extremos locais
1

Calcule fₓ e f_y

Derive parcialmente em x e em y.

2

Resolva fₓ = 0 e f_y = 0 simultaneamente

Encontre todos os pontos (x₀,y₀) que satisfazem o sistema.

3

Calcule as derivadas segundas fₓₓ, f_yy, fₓ_y

4

Calcule D(x₀,y₀) = fₓₓ·f_yy − (fₓ_y)² em cada ponto crítico

5

Aplique o Teorema 2

Consulte a tabela acima para classificar.

6

Calcule o valor f(x₀,y₀) se for extremo

Esse é o extremo local — o valor da função, não o ponto.

Exemplo resolvido — passo a passo
Função a analisar
f(x,y) = x³ + y³ − 3xy

Queremos encontrar todos os extremos locais e pontos de sela desta função.

Passo 1 — Derivadas parciais de primeira ordem
fₓ(x,y) = 3x² − 3y
f_y(x,y) = 3y² − 3x
Passo 2 — Pontos críticos (fₓ = 0 e f_y = 0)
Do sistema:
3x² − 3y = 0 → y = x² ...(1)
3y² − 3x = 0 → x = y² ...(2)

Substitui (1) em (2):
x = (x²)² = x⁴
x⁴ − x = 0
x(x³ − 1) = 0

x = 0 → y = 0 → ponto (0, 0)
x = 1 → y = 1 → ponto (1, 1)
Passo 3 — Derivadas segundas
fₓₓ(x,y) = 6x
f_yy(x,y) = 6y
fₓ_y(x,y) = −3
Passo 4 — Discriminante
D(x,y) = fₓₓ·f_yy − (fₓ_y)²
= (6x)(6y) − (−3)²
= 36xy − 9

Em (0, 0): D = 36·0·0 − 9 = −9
Em (1, 1): D = 36·1·1 − 9 = +27
fₓₓ(1,1) = 6·1 = +6
Passo 5 — Classificação e resultado
PontoDfₓₓClassificaçãoValor f
(0, 0)−9 < 0—Ponto de selaf(0,0) = 0
(1, 1)+27 > 0+6 > 0Mínimo localf(1,1) = −1

✔ Mínimo local = −1, atingido no ponto (1, 1)

Extremos globais

Máximo e mínimo absolutos em domínios compactos

Mínimo global
f(x₀,y₀) ≤ f(x,y), ∀(x,y) ∈ U

O menor valor de f em todo o domínio U.

Máximo global
f(x₀,y₀) ≥ f(x,y), ∀(x,y) ∈ U

O maior valor de f em todo o domínio U.

Teorema de existência Teorema 1

Se f é contínua num conjunto compacto U (fechado e limitado), então f assume seus valores máximo e mínimo globais em U.

✔ Compacto = fechado + limitado. Regiões triangulares, retangulares e circulares com fronteira incluída são compactas.

Onde procurar os extremos

🔵 No interior

Extremo no interior é também local → encontrar pontos críticos (fₓ = f_y = 0) e checar se pertencem ao interior (não à fronteira).

🟠 Na fronteira

Parametrizar cada trecho como g(t) = f restrita ao lado/curva. Procurar g'(t) = 0 no intervalo aberto do trecho.

⚠️ Extremos na fronteira não são locais de f. Não aplique o Teste da Derivada Segunda neles.

⚠️ Não esqueça de avaliar f nos vértices — eles não aparecem nos trechos abertos da fronteira.

Passo a passo — extremos globais em domínio compacto
1

Desenhe e identifique a região R

Marque vértices, equações dos lados e o interior.

2

Interior: resolva fₓ = 0 e f_y = 0

Verifique se cada ponto crítico pertence ao interior (não à fronteira).

3

Fronteira: parametrize cada trecho

Substitua a equação do trecho em f(x,y) → função g(t) de uma variável. Resolva g'(t) = 0 no intervalo aberto do trecho.

4

Avalie f em todos os vértices

5

Compare todos os valores candidatos

O maior é o máximo global; o menor é o mínimo global.

Exemplo — região triangular
f(x,y) = x³ − y³ + 3xy + 3
Vértices: A(1,2), B(1,−2), C(−1,−2)
Interior
fₓ = 3x² + 3y = 0 → y = −x²
f_y = −3y² + 3x = 0 → x = y²

Substituindo: x = (−x²)² = x⁴
x⁴ − x = 0 → x = 0 ou x = 1

(0,0): não pertence ao interior
(1,−1): não pertence ao interior

→ Nenhum candidato no interior
Lado AB — x = 1, y ∈ ]−2, 2[
g(y) = f(1,y) = 1 − y³ + 3y + 3 = −y³ + 3y + 4

g'(y) = −3y² + 3 = 0 → y = ±1

g(1) = f(1,1) = −1+3+4 = 6
g(−1) = f(1,−1) = 1−3+4 = 2
Lado BC — y = −2, x ∈ ]−1, 1[
h(x) = f(x,−2) = x³ + 8 − 6x + 3 = x³ − 6x + 11

h'(x) = 3x² − 6 = 0 → x = ±√2 ≈ ±1.41

Ambos fora do intervalo ]−1,1[ → descartados
Lado AC — y = 2x, x ∈ ]−1, 1[
p(x) = f(x,2x) = x³ − 8x³ + 6x² + 3 = −7x³ + 6x² + 3

p'(x) = −21x² + 12x = 0
x(−21x + 12) = 0
x = 0 ou x = 4/7 → y = 8/7

p(0) = f(0,0) = 3
p(4/7) = f(4/7, 8/7) ≈ 3.65
Resultado final — todos os candidatos
PontoOrigemf(ponto)
(1, 1)Lado AB6
(1, −1)Lado AB2
(0, 0)Lado AC3
(4/7, 8/7)Lado AC≈ 3.65
A(1, 2)Vérticef = 2

✔ Mínimo global = 2

✔ Máximo global = 16

Multiplicadores de Lagrange

Otimização com vínculos (restrições)

Ideia central

Queremos o extremo de f(x,y) restrito ao vínculo g(x,y) = 0. No ponto de extremo, ∇f e ∇g são paralelos:

∇f(x₀,y₀) = λ · ∇g(x₀,y₀)

O escalar λ é o multiplicador de Lagrange.

✔ Geometricamente: curvas de nível de f e a curva g=0 são tangentes no ponto de extremo vinculado.

Passo a passo — 1 vínculo, 2 variáveis
1

Reescreva o vínculo como g(x,y) = 0

Ex.: x² + y² = 1 → g(x,y) = x² + y² − 1

2

Monte o sistema ∇f = λ∇g junto com g = 0

fₓ = λ·gₓ ...(1)
f_y = λ·g_y ...(2)
g(x,y) = 0 ...(3)
3

Elimine λ dividindo (1) por (2)

Válido se λ ≠ 0 e os denominadores ≠ 0. Isso gera uma relação entre x e y.

4

Substitua a relação em (3) para encontrar x e y

5

Verifique casos especiais

Teste λ = 0 e os casos em que denominadores se anulam. Substitua em (3) — se não satisfaz o vínculo, descarte.

6

Compare f em todos os candidatos

O maior é máximo vinculado; o menor é mínimo vinculado.

Dois vínculos — 3 variáveis

Para f(x,y,z) com g(x,y,z) = 0 e h(x,y,z) = 0:

∇f = λ·∇g + μ·∇h

Sistema: 3 equações do gradiente + 2 vínculos = 5 equações para x, y, z, λ, μ.

⚠️ Procure substituições diretas. Ex.: se uma equação do gradiente isola μ imediatamente, use isso primeiro.

Exemplo resolvido — 1 vínculo
f(x,y) = x² + 4y² − x + 2y   sujeito a   x² + 4y² = 1
Montando o sistema
g(x,y) = x² + 4y² − 1

∇f = (2x−1, 8y+2)
∇g = (2x, 8y)

2x−1 = 2λx ...(1)
8y+2 = 8λy ...(2)
x² + 4y² = 1 ...(3)
Eliminando λ — dividir (1)÷(2)
(2x−1)/(8y+2) = (2λx)/(8λy) = x/(4y)

Cruzando:
4y(2x−1) = x(8y+2)
8xy − 4y = 8xy + 2x
−4y = 2x

x = −2y
Substituindo x = −2y em (3)
(−2y)² + 4y² = 1
4y² + 4y² = 1
8y² = 1
y = ±1/(2√2)

y = 1/(2√2) → x = −1/√2 → ponto (−1/√2, 1/(2√2))
y = −1/(2√2) → x = 1/√2 → ponto (1/√2, −1/(2√2))

Caso λ = 0: leva a (1/2, −1/4), não satisfaz (3) → descartado
Caso y = 0: contradição em (2) → descartado
Resultado final
Pontof(ponto)Classificação
(−1/√2, 1/(2√2))1 + √2 ≈ 2,41Máximo vinculado
(1/√2, −1/(2√2))1 − √2 ≈ −0,41Mínimo vinculado

✔ f assume máximo = 1+√2 e mínimo = 1−√2 sobre a elipse x²+4y²=1

Resumo rápido

Tudo num só lugar antes da prova

Extremo local

① Achar pontos críticos
② Calcular D = fₓₓ·f_yy − (fₓ_y)²
③ Classificar: D<0=sela, D>0+fₓₓ>0=mín, D>0+fₓₓ<0=máx

Extremo global

① Interior: pontos críticos
② Fronteira: parametrizar g(t)
③ Avaliar vértices
④ Comparar tudo

Lagrange

① ∇f = λ∇g (+ μ∇h se 2 vínculos)
② Eliminar λ dividindo equações
③ Substituir no vínculo
④ Comparar f nos candidatos

Armadilhas comuns na prova
  • Confundir o extremo (valor f) com o ponto (x₀,y₀) onde ocorre.
  • Aplicar o Teste da 2ª Derivada em pontos da fronteira (só vale no interior).
  • Esquecer de avaliar f nos vértices ao buscar extremos globais.
  • Dividir por zero ao eliminar λ sem verificar o caso λ = 0 separadamente.
  • D > 0 não basta — precisa verificar o sinal de fₓₓ para saber se é máximo ou mínimo.
  • D = 0 → teste inconclusivo. Não classifique como nada!
  • Em Lagrange: candidatos que não satisfazem o vínculo (3) devem ser descartados.
Fórmulas para colar na cabeça
Discriminante:
D(x,y) = fₓₓ · f_yy − (fₓ_y)²

Lagrange — 1 vínculo:
∇f = λ · ∇g

Lagrange — 2 vínculos:
∇f = λ · ∇g + μ · ∇h

Plano tangente em ponto crítico (com derivadas existindo):
z = f(x₀,y₀)   ← horizontal
